Glosario de Programación

30 conceptos esenciales de Python y PHP con un toque visual

Variable
Espacio de almacenamiento en memoria para guardar datos. En PHP se usa $, en Python no se requiere.
Función
Bloque de código reutilizable que realiza una tarea específica. Ayuda a organizar y modularizar el código.
Condicional (if/else)
Estructura que ejecuta código si una condición es verdadera, permitiendo al programa tomar decisiones.
Bucle (for/while)
Estructura que repite un bloque de código. Esencial para procesar listas, arrays o colecciones de datos.
Lista (Python)
Colección de elementos ordenada y mutable. Se define con corchetes `[ ]`.
Array (PHP)
Almacena múltiples valores en una sola variable. Puede ser indexado o asociativo (clave-valor).
Diccionario (Python)
Colección desordenada de pares clave-valor. Permite un acceso eficiente a los datos.
Superglobales (PHP)
Variables integradas que están disponibles en cualquier ámbito del script, como `$_GET`, `$_POST` o `$_SESSION`.
Módulo (Python)
Archivo que contiene un conjunto de funciones, clases y variables. Se importa para usar sus funcionalidades.
Namespace (PHP)
Sistema para encapsular elementos y evitar conflictos de nombres, similar a los módulos en otros lenguajes.
Clase
Una plantilla para crear objetos. Define sus atributos y métodos.
Objeto
Una instancia de una clase. Una entidad concreta creada a partir de una plantilla.
Programación Orientada a Objetos (POO)
Paradigma de programación que organiza el código en objetos que contienen datos y comportamientos.
Herencia
Mecanismo por el cual una clase (hija) adquiere las propiedades y métodos de otra clase (padre).
Polimorfismo
Principio que permite que un mismo método o función se comporte de manera diferente en distintas clases.
Tupla (Python)
Una colección de elementos ordenada e **inmutable**. Se define con paréntesis `( )`.
